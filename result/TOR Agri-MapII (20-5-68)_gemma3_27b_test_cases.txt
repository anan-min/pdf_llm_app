เอกสารที่คุณให้มาเป็นประกาศเชิญชวนประกวดราคาอิเล็กทรอนิกส์ (e-bidding) สำหรับโครงการพัฒนาแพลตฟอร์มแผนที่กลางภาคเกษตรของกรมพัฒนาที่ดิน กระทรวงเกษตรและสหกรณ์ เนื้อหาครอบคลุมรายละเอียดสำคัญต่างๆ ของโครงการ ดังนี้:

**1. วัตถุประสงค์และขอบเขตงาน:**

*   พัฒนาแพลตฟอร์มแผนที่กลางภาคเกษตรเพื่อทดแทนระบบ Agri-Map เดิม
*   เชื่อมต่อกับซอฟต์แวร์ลิขสิทธิ์แผนที่กลาง (Core GIS)
*   รองรับผู้ดูแลระบบและผู้ใช้งานทั่วไปจำนวนมาก
*   เน้นการทำงานบน Web Browser
*   ต้องสอดคล้องกับขอบเขตงานที่กำหนด

**2. เกณฑ์การประเมิน:**

*   **ด้านเทคนิค (40%):** การทดสอบการทำงานของซอฟต์แวร์และแพลตฟอร์มตามที่กรมพัฒนาที่ดินกำหนด
*   **ด้านราคา:** (ไม่ได้ระบุสัดส่วน แต่เป็นส่วนหนึ่งของการพิจารณา)
*   **ด้านอื่นๆ:** (ไม่ได้ระบุรายละเอียด)

**3. คุณสมบัติของผู้เสนอราคา:**

*   (ไม่ได้ระบุรายละเอียดในเอกสารนี้)

**4. ระยะเวลาดำเนินโครงการ:**

*   270 วัน นับจากวันที่ลงนามในสัญญา

**5. งบประมาณโครงการ:**

*   96,464,500 บาท

**6. สถานที่ส่งมอบงาน:**

*   ศูนย์เทคโนโลยีสารสนเทศและการสื่อสาร กรมพัฒนาที่ดิน

**7. ข้อกำหนดอื่นๆ:**

*   การรักษาความลับของข้อมูล
*   ค่าปรับกรณีไม่สามารถทำงานให้แล้วเสร็จตามกำหนด
*   การตัดสินของคณะกรรมการถือเป็นที่สิ้นสุด

**สรุป:**

เอกสารนี้เป็นข้อมูลสำคัญสำหรับผู้สนใจเข้าร่วมประกวดราคาโครงการนี้ ผู้ที่สนใจควรศึกษาเอกสารอย่างละเอียด รวมถึงข้อกำหนดและเงื่อนไขต่างๆ เพื่อให้มั่นใจว่ามีคุณสมบัติครบถ้วนและสามารถปฏิบัติตามข้อกำหนดได้ทั้งหมด

**ประเด็นสำคัญที่ควรทราบ:**

*   **การทดสอบทางเทคนิค (40%):** เป็นส่วนสำคัญที่สุดในการประเมิน ผู้เสนอราคาต้องเตรียมความพร้อมสำหรับการทดสอบอย่างเต็มที่
*   **การเชื่อมต่อกับ Core GIS:** ผู้เสนอราคาต้องมีความเข้าใจและประสบการณ์ในการเชื่อมต่อกับซอฟต์แวร์นี้
*   **การทำงานบน Web Browser:** แพลตฟอร์มและแอปพลิเคชันต้องสามารถทำงานได้อย่างราบรื่นบน Web Browser
*   **การรักษาความลับของข้อมูล:** ผู้เสนอราคาต้องให้ความสำคัญกับการรักษาความลับของข้อมูลอย่างเคร่งครัด

หากคุณมีคำถามเพิ่มเติมหรือต้องการข้อมูลเฉพาะส่วนใด สามารถสอบถามได้เลยครับ
