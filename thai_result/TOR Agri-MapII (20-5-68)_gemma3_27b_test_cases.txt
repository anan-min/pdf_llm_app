เอกสารที่คุณให้มาเป็นเอกสารประกวดราคาอิเล็กทรอนิกส์ (e-bidding) สำหรับโครงการพัฒนาแพลตฟอร์มแผนที่กลางภาคเกษตรของกรมพัฒนาที่ดิน กระทรวงเกษตรและสหกรณ์ สรุปประเด็นสำคัญได้ดังนี้:

**1. วัตถุประสงค์ของโครงการ:**

*   พัฒนาแพลตฟอร์มแผนที่กลางภาคเกษตร เพื่อทดแทนระบบ Agri-Map เดิม
*   เชื่อมต่อกับซอฟต์แวร์ลิขสิทธิ์แผนที่กลาง (Core GIS)
*   รองรับผู้ดูแลระบบและผู้ใช้งานทั่วไปจำนวนมาก
*   เน้นการทำงานบน Web Browser

**2. ขอบเขตงาน:**

*   พัฒนาแพลตฟอร์มและเว็บแอปพลิเคชัน
*   ทดสอบการเชื่อมต่อกับ Core GIS
*   ทดสอบการทำงานบน Web Browser
*   ทดสอบทางเทคนิค (ด้านภูมิสารสนเทศ) ตามที่กรมพัฒนาที่ดินกำหนด

**3. เกณฑ์การประเมิน:**

*   **ด้านเทคนิค (40%):**  ผลการทดสอบทางเทคนิค (ด้านภูมิสารสนเทศ)
*   **ด้านราคา (60%):**  ราคาที่เสนอ

**4. กำหนดการ:**

*   ระยะเวลาดำเนินงาน: 270 วัน นับจากวันที่ลงนามในสัญญา
*   สถานที่ส่งมอบงาน: ศูนย์เทคโนโลยีสารสนเทศและการสื่อสาร กรมพัฒนาที่ดิน

**5. งบประมาณ:**

*   รวมทั้งสิ้น 96,464,500 บาท (เก้าสิบหกล้านสี่แสนหกหมื่นสี่พันห้าร้อยบาทถ้วน)

**6. ข้อกำหนดอื่นๆ:**

*   **การรักษาความลับ:** ผู้ชนะการประกวดราคาต้องรักษาข้อมูลของโครงการและกรมพัฒนาที่ดินเป็นความลับ
*   **ค่าปรับ:** หากไม่สามารถทำงานให้แล้วเสร็จตามกำหนด จะมีค่าปรับ
*   **การตัดสิน:** การตัดสินของคณะกรรมการถือเป็นที่สิ้นสุด

**ประเด็นสำคัญที่ควรทราบสำหรับผู้สนใจเข้าร่วมประกวดราคา:**

*   **ความเชี่ยวชาญด้านภูมิสารสนเทศ:**  โครงการนี้เน้นการทดสอบทางเทคนิคด้านภูมิสารสนเทศเป็นอย่างมาก ดังนั้นผู้เข้าร่วมประกวดราคาควรมีความเชี่ยวชาญในด้านนี้เป็นพิเศษ
*   **ความเข้ากันได้กับ Core GIS:**  แพลตฟอร์มที่พัฒนาขึ้นจะต้องสามารถเชื่อมต่อและทำงานร่วมกับซอฟต์แวร์ Core GIS ได้อย่างราบรื่น
*   **การทำงานบน Web Browser:**  การพัฒนาเว็บแอปพลิเคชันที่สามารถทำงานบน Web Browser ได้อย่างมีประสิทธิภาพเป็นสิ่งสำคัญ
*   **การรักษาความลับของข้อมูล:**  ผู้เข้าร่วมประกวดราคาต้องให้ความสำคัญกับการรักษาความลับของข้อมูลของโครงการและกรมพัฒนาที่ดิน

หวังว่าข้อมูลนี้จะเป็นประโยชน์สำหรับคุณ หากคุณมีคำถามเพิ่มเติม โปรดแจ้งให้ทราบ
